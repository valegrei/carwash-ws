# carwash-ws
RESTful API para Carwash App, usando Node, Express, Mysql y Docker.

## Instalación
Con Docker-Compose, ejecutar en el terminal:
```bash
docker-compose up -d --build
```
Para detener usar:
```bash
docker-compose down
```
Para eliminar todas las imágenes:
```bash
docker-compose down --rmi all
```
